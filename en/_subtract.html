Subtract

The Substract block sets the output to the difference of its two inputs.

f(in1,in2) = in1 - in2

Table 11. Inputs

Name Type Default Description

in1

Double

0.0

The left hand argument in the operation.

in2

Double

0.0

The right hand argument in the operation.


Table 12. Outputs

Name Type Description

output

Double

The difference between the two inputs, output = in1 - in2


Description

in1

The left hand argument of the subtract function.

in2

The right hand argument of the subtract function.

output

The output is set to the difference between in1 and in2, so that output = in1 - in2

Source Code

    output.set( in1.get() - in2.get(), forceOutput );
