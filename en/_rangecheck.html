Table 17. Outputs

Name Type Description

output

Boolean

If the input is within range, then the output is true. Otherwise it is false.


Description

input

The input value to be checked if it is within range.

lowLimit

The lowest value (inclusive) that the input value can be to be considered within range. If the input is lower than this value, the output is false.

highLimit

The highest value (inclusive) that the input value can be to be considered within range. If the input is higher than this value, the output is false.

output

The output is true if the input value is within range, i.e. larger than, or equal to, the lowLimit value AND smaller than, or equal to, the highLimit value. If the input value is not within this range, the output is set to false.

Source Code

    double in = input.get();
    boolean out = in >= lowLimit.get() && in <= highLimit.get();
    output.set( out, forceOutput );
